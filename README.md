# PXL Didactiek Coach
Interactieve web-tool voor studenten van de PXL-lerarenopleiding om lesvoorbereidingen te toetsen aan de hand van het didactisch model.

## Installatie
1. Zorg dat Python geïnstalleerd is.
2. Installeer benodigde bibliotheken: `pip install streamlit google-generativeai`.
3. Stel je Google Gemini API key in via Streamlit secrets.
4. Start de app: `streamlit run main.py`.

## Doel
Begeleiding bij het opstellen van lesvoorbereidingen met feedback op basis van de 12 bouwstenen en Bloom-taxonomie.
